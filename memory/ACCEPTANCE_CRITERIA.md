# 完成验收标准

## 通用完成
1. 不能只口头声称完成，必须有实际工具结果、命令输出或状态查询证据。
2. 文件修改后必须验证语法、引用和运行路径。
3. 清理临时文件、`__pycache__`、旧引用和备份残留。
4. 结果说明必须与实际验证状态一致。

## 媒体任务完成
1. 媒体身份明确，包括类型、年份、TMDB/Douban ID 或路径识别结果。
2. 下载前展示候选并确认，除非用户明确给出链接并要求下载。
3. 添加下载后查询任务状态。
4. 创建订阅后查询订阅状态。
5. 转移后查询转移历史或媒体库存在性。

## 配置/脚本任务完成
1. 编译或语法检查通过。
2. 执行路径验证通过。
3. 引用扫描通过。
4. `agent_self_audit.py` 在需要时通过。

## Job 任务完成
1. `status` 正确。
2. `last_run` 按执行结果更新。
3. recurring 任务保持 `pending`。
4. 执行日志简洁记录结果。
