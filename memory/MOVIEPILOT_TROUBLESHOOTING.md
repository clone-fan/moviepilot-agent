# MoviePilot 故障排查矩阵

MoviePilot matrix：站点 → 识别 → 资源 → 下载 → 转移 → 媒体库 → 通知。

- 搜不到：查站点启用/认证/范围、识别、别名、过滤。
- 下载失败/无速度：查下载器、任务、资源做种、保存路径、站点认证。
- 订阅不更新：查订阅状态、季集、缺集、过滤、站点和媒体库。
- 转移失败：查失败原因、源路径、目标目录、识别、权限；必要时删失败历史再重试。
- 媒体库缺失：查下载完成、转移成功、刮削、命名、媒体服务器同步。
- 通知异常：查绑定、渠道、发送日志、模板数据源。

## 自检锚点
搜不到资源先查站点、识别、过滤；转移失败先查失败历史、源路径、权限和识别。
