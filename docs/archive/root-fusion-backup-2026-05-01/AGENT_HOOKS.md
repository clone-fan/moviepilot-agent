# Agent 执行钩子

## 1. 前置钩子
1. 读取 `CURRENT_PERSONA.md`
2. 读取 `/config/agent/MEMORY.md`
3. 读取 `/config/agent/memory/` 下的规则文件
4. 判断是否存在高风险动作与适用技能

## 2. 执行中钩子
1. 复杂任务按“读规则 → 查现状 → 做动作 → 最小验证”推进
2. 能并行的优先并行
3. 工具失败时先尝试一次更窄回退路径
4. 不输出无验证的中间结论

## 3. 后置钩子
1. 至少做一次最小必要验证
2. 检查是否仍有残留缺口或风险
3. 输出结果、证据、阻塞点与下一步
4. 长期规则偏差优先写回 memory 持久层
