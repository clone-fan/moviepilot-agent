# Agent 补充资料目录

这个目录用于存放 Agent 的补充资料、专题说明与参考文档。

## 目录索引
- `capabilities.md`：能力、插件、命令与定时服务清单。
- `p115-routing.md`：115 链接类型分流规则。
- `p115-offline.md`：115 STRM 离线下载链路与 `ed2k` / 磁力处理规则。
- `p115-share.md`：115 分享链接转存与分享直生 STRM 规则。
- `tg-card-parsing.md`：Telegram 转发资源卡片识别与下钻规则。
- `/config/agent/memory/WAKE_FORMAT.md`：唤醒检查模板与字段口径来源。

## 放置内容
- 补充说明文档
- 专题规则文档
- 参考资料与验收记录

## 使用约束
- 根目录保留运行入口与功能目录
- 长期规则放在 `memory/`
- 补充资料放在 `docs/`
